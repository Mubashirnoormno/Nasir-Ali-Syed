# 🏛️ پروفیسر ناصر علی سید — تصانیف و ادبی پورٹ فولیو
## Professor Nasir Ali Syed — Digital Archives & Portfolio Portal

Welcome to the official digital repository and archive portal of **Professor Nasir Ali Syed**, a distinguished Urdu scholar, presidential award-winning writer, and intellectual from Khyber Pakhtunkhwa, Pakistan. 

This portal serves as an elegant, interactive archive of his lifetime literary achievements, critical publications, Urdu poetry, essays, and historical photo galleries.

---

## 🎨 Design & Aesthetic Excellence
The website is designed with a premium, scholarly parchment look, showcasing:
* **Urdu Typography**: Authentic, high-fidelity **Noto Nastaliq Urdu** & **Gulzar** fonts.
* **Classical Aesthetics**: Deep Mahogany Dark Brown (`#3D1C0B`) paired with warm Amber Gold (`#B45309`) accents on rich Ivory Parchment (`#FCFAF7`).
* **Micro-Animations**: Smooth Framer Motion transitions, physics-based page load screens, and elegant book sleeves hover states.

---

## 🚀 Key Features Included
1. **📚 Library Portal**: Interactive catalog showcasing all his critical and creative volumes (like *Char Su*, *Shamien Farib Dety Hain*, *Adab Ke Atraf Mein*). Includes detailed Urdu synopses, publication years, and English abstracts.
2. **✍️ Ghazals & Poetry**: A beautiful literary canvas showcasing signature couplets in elegant calligraphy layouts.
3. **📸 Biographical Timeline & Media Archive**: Dynamic, categorized photo galleries tracking the professor's interactions with literary legends across the decades.

---

## 💻 Tech Stack
* **Framework**: React (v19) + TypeScript
* **Build Tool**: Vite
* **Styling**: Tailwind CSS (v4)
* **Animations**: Motion (framer-motion)
* **Icons**: Lucide React

## ⚙️ Running Locally

### Prerequisites
* [Node.js](https://nodejs.org/) (v18 or higher recommended)

### Step-by-Step Setup
1. Clone your new repository:
   ```bash
   git clone https://github.com/Mubashirnoormno/Nasir-Ali-Syed.git
   cd Nasir-Ali-Syed
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to:
   ```http
   http://localhost:3000/
   ```
